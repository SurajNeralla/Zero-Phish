# Privacy Policy for Zero Phish Extension

**Last Updated**: December 6, 2024

## Introduction
Zero Phish ("we", "our", or "the extension") is a browser extension designed to detect and protect against phishing websites and malicious redirects. This privacy policy explains how we collect, use, and protect your information.

## Information We Collect

### 1. Browsing Data
- **URLs visited**: We analyze URLs you visit to detect potential phishing sites
- **Page titles**: Collected when you report a phishing site
- **Redirect chains**: Tracked to identify suspicious redirect patterns

### 2. User Reports
When you manually report a phishing site, we collect:
- URL of the reported site
- Page title
- Timestamp of the report
- Browser user agent (for compatibility purposes)

### 3. Technical Data
- Extension version
- Browser type and version
- Operating system (for debugging)

## What We DON'T Collect
- ❌ Personal identification information (name, email, phone)
- ❌ Passwords or login credentials
- ❌ Credit card or payment information
- ❌ Browsing history beyond phishing detection
- ❌ Cookies or tracking data
- ❌ Location data

## How We Use Your Information

### Phishing Detection
- Analyze URLs in real-time to identify phishing attempts
- Track redirect chains to detect malicious patterns
- Build a database of reported phishing sites

### Service Improvement
- Improve detection algorithms
- Fix bugs and enhance features
- Generate anonymous statistics

## Data Storage

### Local Storage
- Extension settings and preferences are stored locally in your browser
- No personal data is stored locally

### Remote Storage (Optional)
If you configure a backend server:
- Phishing reports are sent to your configured backend
- Data is stored in Supabase (third-party database service)
- You control the backend and database

## Data Sharing

### We DO NOT:
- Sell your data to third parties
- Share data with advertisers
- Use data for marketing purposes
- Track you across websites

### We MAY Share:
- Anonymous, aggregated statistics (e.g., "100 phishing sites detected today")
- Reported phishing URLs with security researchers (no personal info)

## Third-Party Services

### Supabase (Optional)
If you configure a backend:
- Data is stored in Supabase databases
- Subject to [Supabase Privacy Policy](https://supabase.com/privacy)
- You control your own Supabase instance

### No Other Third Parties
- No analytics services
- No advertising networks
- No tracking pixels

## Your Rights

### You Can:
- ✅ Disable the extension at any time
- ✅ Delete all stored data by uninstalling
- ✅ Choose not to report phishing sites
- ✅ Run your own backend server
- ✅ Review the source code (open source)

### Data Deletion
- Uninstalling the extension removes all local data
- Backend data can be deleted from your Supabase dashboard

## Security

### How We Protect Your Data:
- HTTPS encryption for all network requests
- No data stored in plain text
- Minimal data collection principle
- Regular security updates

### Limitations:
- We cannot guarantee 100% security
- You are responsible for securing your backend server
- Use strong passwords for admin dashboard

## Children's Privacy
This extension is not intended for children under 13. We do not knowingly collect data from children.
